## 2026-05-05 / 2026-05-06 — about-попап + ревизия дизайн-системы

**Сделано в эту сессию:**
- **Попап «О проекте»** (about-popup): открывается по клику на `i` в шапке. Логотип + текст с ссылкой на ermolaev.space. Использует ту же шторку, что и event-popup (`.popup-overlay`), но с модификатором `.popup--about` и block-layout для надёжного скролла.
- **Дизайн-система — большая ревизия:**
  - Переименование `--text-primary` → `--text-tertiary` (приглушённый); новый `--text-primary` = `--light-100` (контрастный).
  - Все яркие primitives (red/green/blue/yellow/purple) и новый `--toxic` (заменил `--orange`) — теперь полные alpha-шкалы 4..100 (как у white/light/black/dark).
  - `--dark-*` переделан под полупрозрачную шкалу на базе `#3b5172`.
  - Новые токены: `--surface-century` / `--text-century` (на `--red-64`), `--text-secondary` (`--light-64`), `--spacing-3xs` (2px), `--spacing-4xl` (72px).
  - Удалено 5 неиспользуемых semantic-токенов.
  - `--brand` → `#8856fd` (фиолетовый); `--purple-*` база → `#b697ff`; `--yellow-*` → `#feac39`; `--toxic-*` → `#c7ef42`.
  - Все pixel-spacing значения переведены на токены (где совпадают). Несовпадающие обсуждены и заменены на ближайшие.
  - Шрифты: lead 300/34px, h2 300/Light, body 300/26px (теперь все через токены, не хардкод).
- **Утилита типографики** (`js/lib/typography.js`): `fixOrphans` / `fixOrphansInTree` — убирает висячие однобуквенные предлоги в конце строки. Применяется в попапах. Зафиксировано как принцип в проектном CLAUDE.md.
- **Connection lines** переделаны: вертикальные gradient-линии `<div>` (вместо SVG-дуг). Default 1px, hover 3px (растёт симметрично из центра). Центрированы по точкам.
- **Кликабельные имена в «В это же время»**: hover сдвигает на 4px вправо, click открывает попап события этого человека в том же году.
- **Zoom + scroll-hint**: тёмный фон, тонкий padding (2/4 px), pill-shape (`--radius-xxxl`); zoom переехал в левый нижний, scroll-hint в правый.
- **Year hover**: hover в Year scale теперь приобретает размеры Active. Current year overlay: вертикальная линия от курсора + возрасты в пересечениях с life-line. Hover на event-dot включает visual hover линии + цвет категории. Hover на саму life-line подсвечивает только возраст (`--light-32`), без visual hover линии.
- **Чекбоксы в сайдбаре**: активные чекбоксы окрашиваются в цвет своей группы через CSS-переменную `--checkbox-active-color`. События — на `--brand` по дефолту.
- **Раскрывашки токенов/компонентов** в `/library`: блок «Примитивы» теперь сворачиваемый. Под каждым компонентом раскрывашка с зависимостями и токенами (автогенерируется из CSS, см. `js/build/generate-components-meta.js`).

**Не сделано (отложено в BACKLOG):**
- Sticky captions при vertical scroll — требует структурного рефакторинга (вынос captions из event-блоков в отдельный `position: sticky` bar).
- Раскрывашки в библиотеке на каждый variant (а не одна на весь компонент) — парсер должен собирать токены на уровне модификатора.
- Форма обратной связи в about-попапе (план: Formspree, нужна регистрация со стороны пользователя).
